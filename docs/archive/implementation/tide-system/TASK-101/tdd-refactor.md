# TASK-101: 天体計算エンジン - リファクタリング（Refactor Phase）

## 目的
Green Phaseで動作確認できた実装に対して、コードの品質向上と計算精度の改善を行う。
テストが通ることを保ちながら、保守性・パフォーマンス・精度を向上させる。

## リファクタリング目標

### 1. 計算精度の向上
- 月齢計算: ±0.5日 → ±0.1日
- 太陽位置: ±5度 → ±1度
- 月位置: ±5度 → ±2度

### 2. コード構造の改善
- 定数の分離・明確化
- エラーハンドリングの強化
- 計算ロジックの最適化

### 3. パフォーマンス最適化
- 不要な計算の削減
- 数値演算の最適化
